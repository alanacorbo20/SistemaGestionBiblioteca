# **Sistema de Gestión de Biblioteca**



### Descripción



Sistema de Gestión de Biblioteca desarrollado en Java y MySQL como proyecto académico, obteniendo la máxima calificación.



La aplicación permite administrar personas, libros, computadoras, préstamos e ingresos económicos mediante una interfaz gráfica de escritorio desarrollada completamente de forma manual.



Uno de los principales objetivos del proyecto fue implementar una solución completa de gestión, incorporando validaciones, control de préstamos, persistencia de datos y automatización de procesos administrativos.



La interfaz fue construida íntegramente utilizando Java Swing, sin emplear diseñadores visuales ni herramientas de arrastrar y soltar (drag-and-drop), permitiendo un control total sobre el diseño y comportamiento de cada componente.



\---



### Tecnologías Utilizadas



\* Java

\* Java Swing

\* MySQL

\* JDBC

\* Eclipse IDE



\---



### Funcionalidades



##### Gestión de Personas



\* Alta de personas.

\* Modificación de registros.

\* Eliminación de registros.

\* Consulta de información.

\* Validación de cédula uruguaya.

\* Validación de número telefónico.

\* Verificación de campos obligatorios.



##### Gestión de Libros



\* Alta de libros.

\* Modificación de información.

\* Eliminación de registros.

\* Consulta y administración del catálogo.



##### Gestión de Computadoras



\* Alta de equipos.

\* Modificación de registros.

\* Eliminación de equipos.

\* Consulta de disponibilidad.



##### Gestión de Préstamos de Libros



\* Registro de préstamos.

\* Registro de devoluciones.

\* Seguimiento de préstamos activos.

\* Identificación automática de préstamos vencidos.



##### Gestión de Préstamos de Computadoras



\* Registro de préstamos.

\* Registro de devoluciones.

\* Control de préstamos vigentes y vencidos.



##### Control Visual de Estados



El sistema utiliza indicadores visuales para facilitar el seguimiento de los préstamos:



\* Préstamos devueltos resaltados en color verde.

\* Préstamos vencidos resaltados en color rojo.



##### Gestión de Ingresos



\* Registro de montos ingresados.

\* Cálculo automático de la recaudación diaria.

\* Consulta rápida de ingresos registrados.



\---



### Validaciones Implementadas



El sistema incorpora múltiples validaciones para garantizar la integridad de los datos:



\* Validación de cédula uruguaya mediante verificación matemática del dígito de control.

\* Validación de longitud y formato de números telefónicos.

\* Verificación de campos obligatorios.

\* Prevención de registros incompletos o inválidos.

\* Control de consistencia en operaciones de préstamos y devoluciones.



\---



### Características Destacadas



\* Sistema de gestión completo desarrollado desde cero.

\* Interfaz gráfica construida manualmente con Java Swing.

\* Sin utilización de generadores visuales de interfaces.

\* Operaciones CRUD completas en todos los módulos principales.

\* Persistencia de datos mediante MySQL.

\* Validaciones avanzadas para mejorar la calidad de los datos.

\* Control visual del estado de préstamos.

\* Cálculo automático de ingresos diarios.

\* Arquitectura modular orientada a la organización y mantenimiento del código.





\---



### Objetivo del Proyecto



Desarrollar una herramienta que permita gestionar de forma eficiente los recursos de una biblioteca, centralizando la administración de usuarios, materiales, equipos informáticos, préstamos e ingresos económicos en una única plataforma.
